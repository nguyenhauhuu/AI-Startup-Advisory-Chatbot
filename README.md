# AI-Startup-Advisory-Chatbot
Giới thiệu

Dự án này xây dựng chatbot AI hỗ trợ tư vấn khởi nghiệp dựa trên Large Language Models (LLMs) kết hợp với Agentic RAG.
Hệ thống có khả năng:

Trả lời câu hỏi về khởi nghiệp, pháp lý, chính sách

Tra cứu tài liệu (PDF, Word, Web)

Giảm hallucination bằng re-ranking và grading documents

Dự án được thực hiện trong khuôn khổ Khóa luận tốt nghiệp – Ngành Kỹ thuật Phần mềm.
